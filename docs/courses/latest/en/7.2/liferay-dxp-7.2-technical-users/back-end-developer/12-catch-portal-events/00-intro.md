# Module 8: Customize the User Interface

<div class="ahead">
<h4>Learning Objectives</h4>

Understand portal lifecycle and model events, learn how to catch and intercept events and add custom logic, and understand the use cases for leveraging portal events.

<h4>Tasks to Accomplish</h4>
<ul>
    <li>Create a Liferay Module Project using the service template and implement the Service Listener Class</li>
    <li>Create a User Model Listener</li>
</ul>
